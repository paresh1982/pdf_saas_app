# NexGen AI: Project Blueprint (v2.0)

## 🎯 Vision
NexGen is a **Universal Agentic Data Intelligence Platform**. It transforms unstructured PDF documents into high-fidelity structured data based on **Natural Language Instructions**, moving beyond simple invoice reading into a general-purpose enterprise tool.

---

## 🏗️ Technical Architecture

### 1. Frontend (React + Tailwind)
- **Design System:** "Midnight Glassmorphism" (Midnight Black, Electric Violet, Emerald Green).
- **Core Engine:** `react-dropzone` for batch uploads, `framer-motion` for micro-animations.
- **Key Component:** **Agentic Command Center** — A natural language input that steers the extraction logic.

### 2. Backend (Node.js + Express)
- **AI Engine:** Google Gemini 2.5 Pro (Native PDF Multimodal mode).
- **Persistence:** SQLite (`nexgen.db`) for searchable document history.
- **Resilience:** Custom exponential backoff for 429 rate management.

### 3. Verification Layer (The "Auditor")
- **Compliance Logic:** Native HSN-to-GST tax verification.
- **Trust Score:** Calculated (0-100%) based on math accuracy and data completeness.
- **Fraud Detection:** Font and pixel alignment checks (Multimodal).
